# 全国高校起業部 HP制作計画

「放課後NPOアフタースクール」のデザイン構造をベースに、「カタリバ」のローディング演出を組み合わせ、**3つのターゲット（学校・メンター・企業）** に明確に情報を届けるWebサイト構築計画。

---

## コンセプト & デザイン方針

### 1. デザインテーマ：「放課後NPOアフタースクール」完全再現
- **全体構造**: 放課後NPOアフタースクールのUI（ヘッダー、フッター、角丸デザイン）を忠実に再現。
- **配色**: 高校企業部の**オレンジ（#FF6B35）** をメインカラーに据え、信頼感のあるティール、アクセントのイエローで構成。
- **フォント**: 「Zen Maru Gothic」を使用し、親しみやすさを強調。

### 2. 演出：「カタリバ」風ローディング
- **初期表示**: 傾いた白背景 + ロゴ + プログレスバーのアニメーション。

---

## サイトマップ & ページ構成

トップページは情報を詰め込まず、**3つのターゲットに向けた「入り口」** として機能させる。

```
55kigyoubu.com/
├── /              (トップページ - シンプルな誘導)
├── /for-school    (学校の皆様へ - 導入について)
├── /for-mentor    (メンターの皆様へ - 立ち上げ・伴走支援)
├── /for-company   (企業の皆様へ - スポンサー・寄付)
├── /news          (お知らせ)
├── /about         (私たちについて - 理念・法人概要)
└── /contact       (総合お問い合わせ)
```

---

### 1. トップページ (`/`) - Gateway
**コンセプト:**
「君は、すごい経営者になる」という力強いメッセージと共に、高校生・学校・企業をつなぐハブとなるページ。
**構成:**
1.  **カタリバ風ローディング:** ロゴとプログレスバーによる演出。
2.  **ヒーローセクション:**
    *   スライドショー（活動風景、プレゼン、合宿など）。
    *   キャッチコピー：「君は、すごい経営者になる」
3.  **About & Gateway（全国高校起業部とは？）:**
    *   部活動としてビジネスに取り組む「実践型教育」であることを説明。
    *   **3つの入り口（カード形式で誘導）:**
        *   **For School (学校関係者):** 「起業部を導入したい」→ `/for-school`
        *   **For Mentor (メンター):** 「全国に広げたい・伴走したい」→ `/for-mentor`
        *   **For Company (企業・スポンサー):** 「高校生を応援したい」→ `/for-company`
4.  **News:** 最新の活動報告、プレスリリース。
5.  **Footer:** 運営団体情報、リンク。

## 2. ターゲット別ページ詳細

### A. 学校の皆様へ (`/for-school`)
**ターゲット**: 導入を検討する教職員、学校関係者
*   **メッセージ**: 「座学1割・実践9割。学校に『起業部』をつくりませんか？」
*   **コンテンツ**:
    *   導入のメリット（生徒の自主性向上、地域連携）。
    *   カリキュラム概要（100のスキル、4つの武器）。
    *   導入校の活動事例マップ。
    *   導入までの流れ。
*   **CTAボタン**: **「導入相談・資料請求」**
    *   （ボタン色: ティール #38B2AC で信頼感を演出）

### B. メンターの皆様へ (`/for-mentor`)
**ターゲット**: 学校への提案・立ち上げを行い、高校生を伴走支援する大人
*   **メッセージ**: 「あなたの経験が、次世代の翼になる。」
*   **役割**: 
    *   学校への「高校起業部」導入提案。
    *   部活動の立ち上げ支援。
    *   高校生と地元企業をつなぐコーディネート。
    *   活動の伴走・見守り。
*   **コンテンツ**:
    *   メンターのやりがい。
    *   具体的な関わり方のステップ。
    *   現役メンターの声。
*   **CTAボタン**: **「認定メンターにエントリーする」**
    *   （ボタン色: オレンジ #FF6B35 で情熱を演出）

### C. 企業の皆様へ (`/for-company`)
**ターゲット**: 事業趣旨に賛同し、寄付・スポンサーとして支援してくれる企業・個人
*   **メッセージ**: 「日本の未来をつくる高校生を、共に育てませんか？」
*   **コンテンツ**:
    *   スポンサーシップの仕組み（エールシステム）。
    *   ご支援の活用用途（活動費、開発費など）。
    *   パートナー企業ロゴ一覧。
*   **CTAボタン**: **「スポンサーとして応援する」**
    *   （ボタン色: イエロー #FFD700 で光・希望を演出）

## 実装計画

### Phase 1: プロジェクトセットアップ
- Vite + React + Vanilla CSS (手書き風デザイン・アニメーション制御のため)
- ルーティング: React Router (ページ分割のため必須)

### Phase 2: コンポーネント開発
1.  **共通パーツ**: Header, Footer, LoadingScreen
2.  **Topページ**: Hero, NavigationCards (3択誘導), NewsGrid
3.  **下層ページテンプレート**: Hero (小), ContentBlock, CTA Area

### Phase 3: ページ実装
1.  Top Page (`/`)
2.  School Page (`/for-school`)
3.  Mentor Page (`/for-mentor`)
4.  Company Page (`/for-company`)

### Phase 4: レスポンシブ & デプロイ
- モバイルでの「3つの入り口」の見せ方（縦積み）を最適化。
- Cloudflare Pagesへのデプロイ。
